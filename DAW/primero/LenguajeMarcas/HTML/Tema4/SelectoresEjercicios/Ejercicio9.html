El selector adyacente se emplea para seleccionar elementos que son hermanos (su elemento padre es el mismo) y están seguidos en el código HTML. Este selector emplea en su sintaxis el símbolo +. Si se considera el siguiente ejemplo:

Texto con selector adyacente (es h3) es el hermano en primer lugar

Texto con selector adyacente (es h4) al estar en primer nivel se aplica el color rojo de la propiedad h3 + h4

Este segundo elemento <h4> no aparece justo después del elemento <h3> por lo que no cumple la condición del selector adyacente y por tanto no se le aplican los estilos de h3 + h4
